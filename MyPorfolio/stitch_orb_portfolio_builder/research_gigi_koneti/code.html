Neural Weather Prediction

GSoC ’26 Candidate Project

Developing advanced spatiotemporal forecasting models utilizing graph neural networks to predict localized severe weather events. Proposed architecture aims to integrate sparse observational data with global climatological priors to improve short-term prediction accuracy over traditional numerical methods.

2024

Branched Memory Agent (BMA)

Independent Research

Designed and evaluated a novel reinforcement learning agent architecture incorporating a hierarchical, branched episodic memory system. The BMA approach demonstrated improved sample efficiency in sparse-reward environments by structuring past experiences conceptually rather than purely chronologically.
